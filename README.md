## Todo List (Laravel + MySQL)

Single-user, public Todo List app (no login) built with:

- Backend: Laravel (latest)
- Frontend: Blade + Tailwind CSS + Alpine.js
- Database: MySQL

### Features

- Tasks fields: Task Name, Deadline (datetime), Category, Subject, Status (To Do / In Progress / Done)
- Category & Subject management directly from the same UI (modal)
- Visual deadline notifications (Overdue / Due soon within 24h). Badges update every minute (simple JS polling).

### Setup

Prerequisites: PHP, Composer, Node.js, and a running MySQL server.

1) Install dependencies

```bash
composer install
npm install
```

2) Configure environment

```bash
copy .env.example .env
php artisan key:generate
```

Edit `.env` for MySQL:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=tes_daoa
DB_USERNAME=root
DB_PASSWORD=
```

3) Run migrations

```bash
php artisan migrate
```

4) Run the app

```bash
npm run dev
php artisan serve
```

Open: `http://127.0.0.1:8000`

### Main Files

- Routes: `routes/web.php`
- Controllers: `app/Http/Controllers/TaskController.php`, `CategoryController.php`, `SubjectController.php`
- Models: `app/Models/Task.php`, `Category.php`, `Subject.php`
- Migrations: `database/migrations/2026_05_18_00000*_create_*_table.php`
- UI: `resources/views/tasks/index.blade.php`
